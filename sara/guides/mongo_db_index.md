Indexes
-------

Para melhorar o desempenho dos scripts que acompanham o sara pode ser interessante se criar os seguintes indexes:

Tweet id
```
id_str
```

User id
```
user.id_str
```

Nome do usuário
```
user.screen_name
```

Nome do usuário que sofreu o retweet
```
retweeted_status.user.screen_name
```
